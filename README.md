# Keyo Tattoo Studio

Website for **Keyo Tattoo Studio**, a tattoo studio in Locarno (realism, microrealism, blackwork).

## Run locally

This is a static website — no build step needed.

```bash
# from this folder
python3 -m http.server 8080
# then open http://localhost:8080
```

Or just open `index.html` in your browser.

## Pages

- `index.html` — home
- `artist-keyo.html`, `artist-dario.html`, `artist-marta.html` — artists
- `academy.html` — academy
- `css/` — styles · `js/` — scripts · `img/` — gallery

## Tech

Plain HTML, CSS and JavaScript.
